### Суть лабораторной работы
#### Цель: создать простую end-to-end систему, которая генерирует данные, сохраняет их и позволяет анализировать через Redash и Jupyter Notebook.

#### Компоненты системы:
1) **Генератор данных** (скрипт на Python'е) - генерирует данные и добавляет из базу данных (PostgresSQL).
2) **База данных** - хранит сгенерированные данные.
3) **Redash** - dashboard'ы и визуализация.

### Как запустить приложение?
На вашем устройстве должен быть установлен Docker Desktop.

Для работы приложения необходимо склонировать данный репозиторий и перейти в корневую папку проекта:
```
git clone https://github.com/vstaff/data_analysis_lab.git
cd data_analysis_lab
```

Установите необходимые зависимости для работы Python-скрипта (см. ```requirements.txt```) с помощью команды -

```
pip install requirements.txt
```

Далее нужно определить переменные окружения - данные из них используются для развертывания бд и подключения 
к ней. Создайте файл ```.env``` в корне проекта и добавьте необходимые переменные.

Пример того как должны выглядеть переменные окружения можете посмотреть в корне проект в файле ```.env.example```. 
Можно просто скопировать данные и вставить их в ```.env```.

Далее необходимо запустить базу данных и Redis:
```docker-compose up -d db redis```

Подождите некоторое время пока база запустится. Проверить можно следующим образом:
``` 
docker-compose ps
```

Инициализируйте таблицы Redash:
```
docker-compose run --rm redash-server create_db
```

Запустите все сервисы:
```
docker-compose up -d
```

### Настройка Redash.
1) Откройте в браузере http://localhost:5000
2) Создайте аккаунт администратора (данные любые на ваше усмотрение)

### Подключение Data Source.
1) Перейдите в настройки Settings → Data Sources → New Data Source
2) Выберите базу данных PostgresSQL
3) Заполните форму следующим образом:

| **Параметр**  | **Значение** |
|---------------|--------------|
| Name          | Main_DB      |
| Host          | db           |
| Port          | 5432         |
| User          | admin        |
| Password      | 12345        |
| Database Name | online_shop  |

4) Нажмите save
5) Проверьте соединение - нажмите **Test Connection** - должно появиться **Success**
